# Smart-Home-IOT-System
Designed a Smart Home IoT Control System that enables users to manage connected devices, monitor home activities, and automate routines through a seamless and intuitive user interface.

## Figma Prototype Link:-https://www.figma.com/design/JAdzmxyNCxEOklxONvpQ5f/Untitled?node-id=2008-91&t=aCoiToNjwRSQHuGK-1

## Overview

The Smart Home IoT Control System is a user-centered mobile application designed to simplify the management of connected smart home devices. The platform enables users to monitor, control, and automate various home appliances through a centralized and intuitive interface, enhancing convenience, efficiency, and overall user experience.

## Problem Statement

Managing multiple smart devices across different applications can create a fragmented and confusing experience for users. This project addresses the challenge by providing a unified platform that allows seamless control and monitoring of all connected devices from a single dashboard.

## Solution

The proposed solution offers a streamlined and accessible interface where users can manage smart devices, create automation routines, monitor device status in real time, and receive important notifications. The design prioritizes usability, clarity, and efficient task completion while minimizing cognitive load.

## Key Features

* Centralized Smart Device Management
* Real-Time Device Monitoring
* Room-Based Device Organization
* Custom Automation and Scheduling
* Energy Usage Insights
* Security and Alert Notifications
* Personalized User Settings

## UI/UX Focus

The design emphasizes a clean visual hierarchy, intuitive navigation, accessibility, and consistency across all screens. Special attention was given to reducing user effort, improving discoverability of controls, and creating a seamless experience for both new and experienced smart home users.

## Tools Used

* Figma
* Design Systems & Components
* Prototyping and User Flow Design

## Outcome

The final design delivers a modern and efficient smart home management experience, enabling users to interact with connected devices effortlessly while maintaining control, visibility, and convenience within a single platform.
